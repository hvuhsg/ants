# ants
Allow you to create you own custom decentralize job management system.

## Install
```shell script
$> git clone https://github.com/hvuhsg/ants.git
```


## Run monitor example
```shell script
$> cd ants/examples/monitor
$> python3 -m venv venv
$> . venv/bin/activate
$> pip install -r requirements.txt
$> pip install ../../.
$> python run.py
```
